Financial Planning Services

We utilize financial planning software called MoneyGuide Pro. This software is a state of the art planning program that collects all of the client's financial data and forecasts the quickest path to retirement. Financial plans can start at $200 and they include the follow analysis:


Financial Goal Plan


The Financial Goal Plan is the most robust and popular Plan type available. The Financial Goal Plan creates a Retirement Plan. It also runs Monte Carlo analysis to determine the Probability of Success of funding all goals entered in the Plan. The Financial Goal Plan is the only Plan type that allows you to create multiple Goals, model retirement income sources (i.e. Social Security and Pension) that will be available to fund Goals, and create alternate scenarios to determine which set of variables produces the most successful Plan. An insurance analysis and an estate analysis are also available within this Plan type.


Asset Allocation Plan


The Asset Allocation Plan runs a Monte Carlo analysis to determine if your current allocation or a target portfolio would provide a greater hypothetical future value. The Asset Allocation Plan does not use income sources such as Social Security and pension income. If you would like to include retirement income sources, use the Financial Goal Plan.


Lifetime Income Plan


The Lifetime Income Plan runs a Monte Carlo analysis to determine if your reallocated portfolio is able to distribute a desired dollar amount over your lifetime. The Lifetime Income Plan does not use income sources such as Social Security and pension income. If you would like to be able to compare side by side withdrawal amounts or include retirement income sources, use the Financial Goal Plan In addition, the client may choose additional analyses. In order to provide such services and meet our fiduciary responsibilities to the Client, it is necessary for the Client to provide information to Copper Canyon about their financial situation, financial goals and risk tolerance.
